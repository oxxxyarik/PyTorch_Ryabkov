## Project Description
Решение содержит модификации базовых моделей линейной и логистической регрессии на PyTorch, кастомные механизмы загрузки данных и скрипты для проведения экспериментов.

## Structure
* `homework_model_modification.py` — Реализация класса `EarlyStopping`, функций подсчета штрафов `L1` / `L2` регуляризации, а также модели многоклассовой логистической регрессии `MulticlassLogisticRegression` с расчетом расширенных метрик Precision, Recall, F1, ROC-AUC и сохранением графиков матрицы ошибок.
* `homework_datasets.py` — Кастомный класс `CSVDataset` для автоматического считывания `.csv` файлов, включающий One-Hot Encoding для текстовых категориальных признаков и Мин-Макс нормализацию числовых колонок перед конвертацией в тензоры.
* `homework_experiments.py` — Скрипт генерации нелинейных данных, создания полиномиальных признаков второй степени и проведения Grid Search для поиска оптимальных гиперпараметров (`learning_rate` и коэффициентов регуляризации).

## Generated Artifacts
